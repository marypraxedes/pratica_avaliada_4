# 📚 Prática Avaliada 4 - Banco de Dados

Este repositório contém a resolução da **Prática Avaliada 4** do **Bootcamp Java Full Stack da Generation Brasil**, desenvolvida utilizando **SQL** e **MySQL**.

O objetivo da atividade foi praticar a modelagem de bancos de dados relacionais, criação de tabelas, manipulação de dados e consultas utilizando os principais comandos da linguagem SQL.

## 🎯 Objetivos

Durante a prática foram realizados:

- Criação de bancos de dados;
- Criação de tabelas com chaves primárias e estrangeiras;
- Inserção de registros (`INSERT`);
- Consultas utilizando `SELECT`;
- Filtros com `WHERE`;
- Atualização de registros (`UPDATE`);
- Exclusão de registros (`DELETE`);
- Relacionamento entre tabelas utilizando `INNER JOIN`.

## 📂 Exercícios Desenvolvidos

### 🎓 Gestão de Cursos
- Cadastro de cursos;
- Cadastro de alunos;
- Relacionamento entre alunos e cursos.

### 🏢 Empresa
- Cadastro de departamentos;
- Cadastro de funcionários;
- Relacionamento entre funcionários e departamentos.

### 🛍️ Loja
- Cadastro de categorias;
- Cadastro de produtos;
- Relacionamento entre produtos e categorias.

## 🛠️ Tecnologias Utilizadas

- SQL
- MySQL
- MySQL Workbench

## 📚 Bootcamp

Desenvolvido como parte das atividades do **Bootcamp Java Full Stack** da **Generation Brasil**, com foco na aprendizagem de banco de dados relacionais e na linguagem SQL.
